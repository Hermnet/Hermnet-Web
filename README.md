# Hermnet Web

Este repositorio contiene la página web de presentación para **Hermnet**, una aplicación de mensajería Zero-Knowledge diseñada con privacidad absoluta y cifrado de extremo a extremo.

## 🌐 Enlaces del Proyecto

- **Página Web:** [https://hermnet.github.io/Hermnet-Web/](https://hermnet.github.io/Hermnet-Web/)
- **Repositorio Original (App Hermnet):** [https://github.com/Hermnet/Hermnet](https://github.com/Hermnet/Hermnet)

## 💻 Tecnologías Utilizadas

La landing page ha sido desarrollada buscando un diseño moderno, dinámico y de alto rendimiento utilizando las siguientes tecnologías:

- **HTML5:** Estructura semántica de la página.
- **CSS3 (Vanilla):** Diseño responsivo, animaciones fluidas (keyframes, transformaciones), variables CSS y efectos visuales avanzados (como desenfoques, sombras dinámicas y gradientes) sin depender de frameworks pesados.
- **React 18:** Integrado directamente para componentes interactivos específicos.
- **Google Fonts:** Uso de tipografías modernas como *Bricolage Grotesque* (para títulos y componentes de gran tamaño) y *Plus Jakarta Sans* (para lectura fluida).
- **GitHub Pages:** Despliegue estático, rápido y gratuito directamente desde la rama `main`.

## 🚀 Despliegue

La página web está configurada para desplegarse automáticamente a través de **GitHub Pages**. El archivo principal, `index.html`, junto con las carpetas de recursos (`assets`, `uploads`), se encuentran en la raíz del repositorio para garantizar un funcionamiento e indexación correctos.
